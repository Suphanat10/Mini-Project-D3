1. กราฟแสดงการกระจายของ superhero

กราฟแสดงการกระจายของนักฮีโร่ที่มีการแบ่งแยกตามเพศ กราฟแสดงการกระจายของความสูงและน้ำหนักของนักฮีโร่ที่มีการแบ่งแยกตามเพศ โดยแกน x จะแสดงข้อมูลน้ำหนัก (kg) และแกน y จะแสดงข้อมูลความสูง (cm) ของนักฮีโร่
โดยจะมีการใช้สีสำหรับแสดงเพศของนักฮีโร่โดยใช้สีน้ำเงินสำหรับเพศชาย สีแดงสำหรับเพศหญิง และสีเทาสำหรับค่าอื่น ๆ นอกจากนี้ ยังสร้าง UI Control เพื่อให้ผู้ใช้สามารถเลือกข้อมูลด้วยการลากแถบบนกราฟได้
โดยเมื่อเลือกข้อมูลแล้วจะมีการเปลี่ยนสีของจุดที่ตรงกับช่วงที่ถูกเลือก นอกจากนี้ยังมีการเพิ่ม slider เพื่อให้ผู้ใช้สามารถเลือกความสูงขั้นต่ำที่จะแสดงบนกราฟได้ด้วย

2. กราฟแสดงการแบ่งแยกจำนวน superhero ตามเพศเป็นรูปแบบของ Pie Chart

กราฟแสดงการแบ่งแยกจำนวนนักฮีโร่ตามเพศเป็นรูปแบบของ Pie Chart มีการนับ เพื่อนับจำนวนนักฮีโร่ตามเพศและแบ่งกลุ่มข้อมูลออกเป็น slice ของ Pie Chart เเละ ยังมีการเพิ่ม Legend เพื่อแสดงค่าต่าง ๆ ใน Pie Chart
โดยการสร้าง rect สำหรับแสดงสีของแต่ละ slice และ text สำหรับแสดงชื่อของแต่ละ slice นอกจากนี้ ยังมีการเพิ่ม checkbox เพื่อให้ผู้ใช้สามารถซ่อนแสดง slice ตามเพศของนักฮีโร่ได้ด้วย

3. กกราฟแบบ Grouped Bar Chart

แสดงผลสรุปสถิติความสูงและน้ำหนักของฮีโร่ตามสีผม โดยมีการเพิ่ม interaction กับ UI control โดยใช้ฟังก์ชัน ในการเลือกแสดงข้อมูลแบบกราฟแท่ง Grouped Bar Chart โดยเลือกตามสีผมที่ต้องการ
โดยจะมีการอัพเดทการแสดงผลกราฟเมื่อมีการเปลี่ยนค่าสีผมที่เลือกใน UI control ด้วย
