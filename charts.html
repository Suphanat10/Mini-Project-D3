4. กราฟแท่งแสดงจำนวนของซูเปอร์ฮีโร่ตามสีตา

กราฟแท่งแสดงจำนวนของซูเปอร์ฮีโร่ตามสีตา และกราฟถูกสร้างขึ้นเรียบร้อยแล้ว จะมี UI control เป็น drop-down menu ที่มีตัวเลือก "All" และ "Male" หรือ "Female" เพื่อกรองข้อมูลและแสดงแผนภูมิแท่งเฉพาะเพศที่เลือกโดยกราฟแท่งจะเปลี่ยนสีเมื่อเมาส์ชี้ไปที่แท่ง และกลับมาเป็นสีเดิมเมื่อเมาส์ออกไป
